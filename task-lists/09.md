# Task 1 – Conceptual questions

Answer in your own words:

1. What is the difference between **batch** learning and **stream** learning?
2. Explain **sliding window** vs. **decaying weights** for summarizing recent data.
3. Why is **sampling** a stream non-trivial compared to sampling a fixed file?
4. What is **concept drift** and give **one** business example.

---

# Task 2 – Stream summary sketch (pen-and-paper)

A sensor emits integers **one per second**. You only have memory for **3** numbers besides the current count.

1. Propose how you would maintain an **approximate count** of distinct values (Flajolet–Martin level idea acceptable in words) **or** a **moving average** of the last **10** values—pick **one** and describe updates when a new value arrives.
2. What fails if the stream rate **doubles** but memory stays fixed?
