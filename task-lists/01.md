# Task 1 – Conceptual Questions

Answer the following questions in your own words.

1. What is **association rule mining** and what types of problems can it solve?
2. What is the difference between:

   * frequent itemsets
   * association rules
3. Explain the following measures:

   * support
   * confidence
   * lift
4. Why are these metrics important in evaluating association rules?
5. Provide at least **two real-world examples** where association rule mining can be applied.

# Task 2 – Transaction Dataset Analysis

Consider the following transaction dataset:

| Transaction | Items               |
| ----------- | ------------------- |
| T1          | bread, milk         |

| T4          | bread, milk, butter |
| T5          | milk                |
| T6          | bread, milk         |
| T7          | butter              |
| T8          | bread, milk, butter |

### Tasks

1. Convert the dataset into **binary representation**.
2. Identify all **frequent itemsets** for:

```
minimum support = 40%
```

4. Generate at least **three possible association rules** from the dataset.
