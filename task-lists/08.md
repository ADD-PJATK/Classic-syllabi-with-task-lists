# Task 1 – Conceptual questions

Answer in your own words:

1. Why can **sorting all records by every attribute** (as in early scalable trees) become a bottleneck?
2. What is the difference between **pre-sorting** strategies (high-level idea of SLIQ-style) and **in-memory resident** strategies?
3. What is a **Hoeffding bound** and how does **VFDT** use it to choose a split **before** seeing all data?
4. What does **concept drift** mean for a tree that is trained incrementally?

---

# Task 2 – Compare batch vs. streaming mindset

For a dataset with **numeric** feature `x` and binary label `y`:

1. Describe how **one** batch algorithm (e.g. CART-style greedy split on `x`) would pick a threshold using **all** training rows.
2. Describe in **5–7** sentences how **VFDT** might approximate that split using **batches of arriving rows** and the Hoeffding idea (no need to derive formulas—explain intuition).
