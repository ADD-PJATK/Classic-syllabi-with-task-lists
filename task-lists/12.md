# Task 2 – Centrality comparison

On a **small undirected** graph of **your choice** (≥ **6** nodes, ≥ **8** edges, cite if from literature):

1. Compute **degree centrality** for all nodes (normalize if you wish—state formula).
2. Compute **betweenness** or **closeness** for at least **3** nodes using a tool (NetworkX, Gephi export, etc.)—state which centrality.
3. Which node is most “important” under each measure? Any disagreement?
