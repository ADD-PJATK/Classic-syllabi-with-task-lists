---

# 2. Basic Concepts

**Collaborative filtering:**

* **User-based:** find users similar to the target user; recommend items those similar users liked.
* **Item-based:** find items similar to the ones the user liked; recommend those items.

**Rating matrix:** rows = users, columns = items, values = ratings (or implicit feedback).

**Content-based:** use item features (e.g. genre, tags) and user preferences to recommend similar items.

---

# 3. Data Format

Typical format: **user_id**, **item_id**, **rating** (and optionally timestamp).

Example:

| user_id | item_id | rating |
|---------|---------|--------|
| 1       | 101     | 5      |
| 1       | 102     | 3      |
| 2       | 101     | 4      |

# 4. Implementing a Recommender in RapidMiner

### Step 1 – Load rating data

Use **Read CSV** (or Excel) with columns: user, item, rating. Set roles (e.g. id, label).

### Step 2 – Split data

Use **Split Data** (e.g. 80% training, 20% testing) with stratified sampling if needed.

### Step 3 – Recommender operator

RapidMiner has **Recommendation** extension or similar operators:

* **Collaborative Filtering** (user-based or item-based).
* Set parameters (e.g. number of neighbors, similarity measure).

### Step 4 – Apply and predict

Apply the model on the test set and compare predicted vs. actual ratings.

# 5. Example Task (Instructor Demonstration)

Using a small rating dataset (e.g. 20 users, 50 items, 200 ratings):

1. Load data and split into train/test.
2. Run a **user-based** or **item-based** collaborative filtering model.
3. Show **predicted ratings** for a few user–item pairs.
4. Compute **MAE** or **RMSE** on the test set.

# 6. Student Tasks

## Task 1 – Get rating data

Obtain a small rating dataset (e.g. MovieLens 100K, or a subset). Load it into your tool and describe the format (number of users, items, ratings, scale).

---

## Task 2 – Build a recommender

1. Split data into training and test sets (e.g. 80/20).
2. Train a **collaborative filtering** model (user-based or item-based).
3. Generate **predictions** for the test set.

---

## Task 3 – Evaluate

1. Compute **MAE** and/or **RMSE** on the test set.
2. For 2–3 users, list **top-5 recommended items** they have not yet rated.
3. Briefly comment: do the recommendations seem reasonable?
