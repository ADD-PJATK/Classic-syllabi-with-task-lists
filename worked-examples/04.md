# 2. Basic Concepts

**PCA (Principal Component Analysis):**

* Finds directions of **maximum variance** in the data.
* Projects data onto fewer dimensions (principal components).
* Reduces noise and redundancy.

**SVD (Singular Value Decomposition):**

* Matrix factorization: \( X = U \Sigma V^T \).
* Used in PCA and in many recommender systems (matrix completion).

**EigenFace:**

* Treat each face image as a long vector (e.g. 64×64 → 4096 dimensions).
* Apply PCA on the set of face vectors.
* Principal components are “eigenfaces”; new faces can be represented by few coefficients.

---

# 3. Feature selection vs. feature extraction

* **Selection:** choose a subset of existing attributes (e.g. filter methods, wrappers).
* **Extraction:** build new features from original ones (e.g. PCA creates linear combinations).

---

# 4. Loading image data

* Images are typically stored as **one row per image**, each column = one pixel (or flatten a 2D image to 1D).
* Grayscale values (e.g. 0–255) should be **normalized** (e.g. 0–1 or standardised).
