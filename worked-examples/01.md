# 2. Basic Concepts

Association rule mining discovers relationships between items in large datasets.

Example rule:

```
{bread, butter} → {milk}
```

Meaning:

Customers who buy **bread and butter** are likely to also buy **milk**.

---

## Important Metrics

### Support

Support measures how frequently a rule appears in the dataset.

Formula:

```
support(A → B) = transactions containing A and B / total transactions
```

Example:

If 20 out of 100 transactions contain **bread and milk**, then:

```
support = 20%
```

---

### Confidence

Confidence measures how often B appears when A appears.

Formula:

```
confidence(A → B) = transactions containing A and B / transactions containing A
```

Example:

If 50 customers buy bread and 20 of them also buy milk:

```
confidence = 20 / 50 = 40%
```

---

### Lift

Lift measures the strength of a rule compared to random occurrence.

```
lift(A → B) = confidence(A → B) / support(B)
```

Interpretation:

* lift > 1 → positive association
* lift = 1 → independent
* lift < 1 → negative relationship

# 4. Loading a Dataset

For association rule mining we typically use **transactional datasets**.

Example dataset:

| Transaction ID | Items               |
| -------------- | ------------------- |
| 1              | bread, milk         |
| 2              | bread, butter       |
| 3              | milk, butter        |
| 4              | bread, milk, butter |
| 5              | milk                |

# 5. Example Task (Instructor Demonstration)

Dataset:

| Transaction | Items               |
| ----------- | ------------------- |
| T1          | bread, milk         |
| T2          | bread, butter       |
| T3          | milk, butter        |
| T4          | bread, milk, butter |
| T5          | milk                |

### Task

Find association rules with:

```
minimum support = 40%
minimum confidence = 60%
```

### Step 1

Convert dataset into binary representation:

| Transaction | Bread | Milk | Butter |
| ----------- | ----- | ---- | ------ |
| T1          | 1     | 1    | 0      |
| T2          | 1     | 0    | 1      |
| T3          | 0     | 1    | 1      |
| T4          | 1     | 1    | 1      |
| T5          | 0     | 1    | 0      |

---

### Step 2

Identify frequent itemsets.

Example:

```
{milk} support = 4/5 = 80%
{bread} support = 3/5 = 60%
{butter} support = 3/5 = 60%
```

---

### Step 3

Generate rules.

Example rule:

```
bread → milk
```

Calculate:

```
confidence = support(bread ∧ milk) / support(bread)
```
