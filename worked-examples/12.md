# Exercise List 12

**Course:** Analysis of Large Data Sets (ADD)  
**Laboratory Topic:** Social and web graphs – node-, group-, network-, hierarchy-centric views  
**Tools:** Python `networkx` / Gephi / Neo4j (Cypher basics, optional)

---

# 1. Goal of the Laboratory

Students will:

* use **centrality** measures at **node** level  
* interpret **community structure** as a **group-centric** view  
* name **global** network statistics (network-centric)  
* see **hierarchies** (e.g. trees, nested groups) as a separate lens  

---

# 2. Basic Concepts

**Node-centric:** degree, betweenness, closeness, eigenvector centrality; **egonet** = neighborhood subgraph of one node.

**Group-centric:** **community detection** (modularity clustering, label propagation, …); dense internal edges vs. sparse cuts.

**Network-centric:** diameter (approx.), average path length, global clustering coefficient, connected components.

**Hierarchy-centric:** organizational charts, **spanning trees**, URL directory trees, taxonomies—often **directed** and **acyclic**.
