# 2. Basic Concepts

**PageRank (directed graph):**

* With probability **d**, follow a random outgoing edge; with **(1−d)**, “teleport” uniformly.  
* Pages with many **in-links** from strong pages score higher.

**HITS:**

* **Authority:** well-cited by good **hubs**.  
* **Hub:** points to many good **authorities**.  
* Iterates **hub** and **authority** scores with normalization.

**Why iterate?** One step is not enough to propagate global importance; power iteration (PageRank) or alternating HITS updates converge under usual conditions.

```python
import networkx as nx
G = nx.DiGraph()
G.add_edges_from([("A", "B"), ("A", "C"), ("B", "C"), ("C", "B")])
nx.pagerank(G, alpha=0.5)
```

# 4. HITS on the same graph
