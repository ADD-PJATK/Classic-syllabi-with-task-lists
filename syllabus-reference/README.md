# Syllabus reference

`ADD_stacjonarne.pdf` — official **Polish** module syllabus (*Analiza dużych zbiorów danych / ADD*, stationary second-degree programme excerpt) used as the source for lecture/lab topic order in this repository.  

Task lists and `project.md` are **English** teaching materials aligned with those topics; if discrepancies arise, the accredited syllabus at PJATK takes precedence.
